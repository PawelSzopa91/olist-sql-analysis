# Olist Brazil E-commerce SQL Analysis 📊

## Overview
Analysis of 230k+ records from Brazil's largest marketplace. The project focuses on scaling, logistics, and customer geography using PostgreSQL.

## Business Insights
* **Top Market**: São Paulo (SP) dominates with **15,540 customers**, followed by Rio de Janeiro (6,882).
* **Delivery Excellence**: São Paulo also boasts the fastest delivery time at **8.3 days**, compared to other regions.
* **Growth Peak**: Significant revenue spike in Nov 2017 (**1.15M BRL**).

## Tools
* **Database**: PostgreSQL
* **Client**: DBeaver
